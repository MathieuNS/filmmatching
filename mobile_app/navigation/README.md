# `navigation/`

Circulation entre les écrans avec **React Navigation** (remplace `react-router-dom`).

À venir en **Phase 3** :
- `AuthStack` — écrans hors connexion (landing, login, inscription, mots de passe).
- `AppStack` — écrans une fois connecté.
- `HamburgerMenu` — menu burger porté du web (Swipe / Liste / Amis / Compte + pages secondaires).
- `RootNavigator` — choisit AuthStack ou AppStack selon `authSlice`.
